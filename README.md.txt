Sales Performance Dashboard (Power BI)

Project Overview

This project analyzes sales performance data to identify trends, profitability patterns, and customer segment performance using interactive Power BI dashboards.

The objective is to transform raw sales data into meaningful business insights that support data-driven decision-making.


Dataset Description

The dataset contains sales transactions including:

- Order Date
- Customer Information
- Product Details
- Sales
- Quantity
- Discount
- Profit
- Region
- Category
- Customer Segment


Tools Used

- Microsoft Excel
  - Data cleaning
  - Data preparation

- Power BI
  - Data visualization
  - Interactive dashboard creation

- DAX
  - KPI calculations
  - Business measures


Key Performance Indicators (KPIs)

The dashboard tracks:

- Total Sales
- Total Profit
- Total Orders
- Profit Margin
- Total Quantity Sold

Key Insights

- Top-performing categories contribute significantly to overall revenue.
- Sales volume does not always translate into higher profitability.
- Customer segments demonstrate different purchasing behaviors.


Recommendations

- Focus marketing efforts on high-performing categories.
- Review discount strategies to protect profit margins.
- Develop targeted strategies for different customer segments.
